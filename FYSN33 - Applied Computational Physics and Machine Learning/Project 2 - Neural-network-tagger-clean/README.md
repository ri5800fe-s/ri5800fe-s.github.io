# Project 2 - Neural-Network Tagger for LHC
by Aditya Majumdar, Réka Kovács and Ricardo Fernandes

## Overview
The code is written in a Jupyter Notebook. Inside the folder we have 5 folders:
- data: jets.csv and pythia.csv;
- extras: unchanged;
- notebooks: wth the notebook that runs everything;
- outputs: plots produced;
- src: python file with accessory functions.

Outside the folder, we have some files unchanged.
## Instalation

All packages needed are written in the requirments.text. You can also just install scikit-learn by just running the fist notebook cell.


## Usage
Pull the folder and start running the Jupyter notebook (Project2.ipynb) in the folder notebooks.
